# Machine Learning Visualizations

Interactive visualizations for teaching machine learning concepts, developed at AImotion Bavaria (Technische Hochschule Ingolstadt).

## üìä Probability Theory

Interactive demonstrations of fundamental probability distributions and concepts:

- [Bernoulli Distribution](https://loma-aimotion.github.io/ml_visualizations/bernoulli-plotly.html) - Interactive Bernoulli distribution visualization
- [Bernoulli Distribution (Bar Chart)](https://loma-aimotion.github.io/ml_visualizations/bernoulli-bar.html) - Alternative bar chart representation

- [Binomial Distribution](https://loma-aimotion.github.io/ml_visualizations/binomial-dist.html) - Interactive binomial distribution
- [Discrete Uniform Distribution](https://loma-aimotion.github.io/ml_visualizations/discrete-uniform.html) - Discrete uniform distribution visualization
- [Continuous Uniform Distribution](https://loma-aimotion.github.io/ml_visualizations/continuous-uniform.html) - Continuous uniform distribution
- [Parametric Distributions](https://loma-aimotion.github.io/ml_visualizations/parametric-dist-plotly.html) - Exploration of parametric distribution families

## üìà Statistics

Visualizations for statistical inference and sampling concepts:

- [Sampling with Confidence Intervals](https://loma-aimotion.github.io/ml_visualizations/sampling-plotly_CI.html) - Confidence interval construction
